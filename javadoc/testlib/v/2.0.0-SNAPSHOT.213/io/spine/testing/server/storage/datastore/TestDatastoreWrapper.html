         static TestDatastoreWrapper wrap(Datastore datastore, boolean waitForConsistency)

        Wraps a given Datastore.

        The waitForConsistency parameter allows to add a delay to each write operation to compensate for the eventual consistency of the storage.

        The waitForConsistency parameter should usually be set to false when working with a local Datastore emulator.
