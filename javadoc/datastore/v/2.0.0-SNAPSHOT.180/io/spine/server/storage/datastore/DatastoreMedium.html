      • read

         abstract <R> DsQueryIterator<R> read(StructuredQuery<R> query)

        Queries the Datastore with the given arguments.

        The Datastore may return a partial result set, so an execution of this method may result in several Datastore queries.

        The limit included in the StructuredQuery, will be a maximum count of objects in the returned iterator.

        The returned DsQueryIterator allows to create a query to the next page of results reusing an existing cursor.

        The resulting Iterator is evaluated lazily. A call to Iterator.remove() causes an UnsupportedOperationException.

        Parameters:
        query - Query to execute upon the Datastore
        Returns:

        results fo the query as a lazily evaluated Iterator
