Package io.spine.server.storage.datastore.config

This package determines how the Datastore-backed storage implementations may be configured.

See: Description

  • Class Summary 
    Class Description
    RecordLayoutDescribes the layout of Datastore Entities in which the Message records of certain type are stored.
    EntityGroupLayoutDescribes the structure of Datastore Entity groups in which the Message records served by a certain RecordStorage are persisted.
    DsColumnMappingThe default Datastore column mapping.
    FlatLayoutDescribes the type of storage layout, in which the records are stored in a flat structure.
    BuilderA builder of StorageConfiguration instances.
    StorageConfigurationA configuration of a Datastore-backed record storage.
  • Interface Summary 
    Interface Description
    CreateStorageSupplies the custom implementation of the record storage.
    CreateRecordStorageA function to create a storage for plain Message records.
    CreateEntityStorageA function to create a storage for Entity states.
