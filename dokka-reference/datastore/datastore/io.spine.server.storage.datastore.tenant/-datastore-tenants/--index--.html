public static TenantIndex index(Datastore datastore, NsConverterFactory converterFactory)

Creates a TenantIndex for the given Datastore.

In a multitenant application it is necessary to pass an instance of the TenantIndex to a BoundedContextBuilder when creating an instance of BoundedContext.

An example of creating a multitenant BoundedContext with the tenant index built on top of Datastore: 


    Datastore datastore = myDatastoreOptions.getService();

    // Configure the `ServerEnvironment` with the factory on top of the Datastore instance.
    // ...

    // Build the `TenantIndex` on top of the same Datastore instance.
    TenantIndex myTenantIndex = DatastoreTenants.index(datastore);


    BoundedContext context = BoundedContext.multitenant("Code samples")
                                           .setTenantIndex(myTenantIndex)
                                           .build();

A single-tenant app (or a single-tenant BoundedContext in a multitenant app) does not require a TenantIndex to be set explicitly, so this method assumes that it is an a single tenant context.
