DsQueryIterator

public final class DsQueryIterator<R>

An Iterator over the com.google.cloud.datastore.StructuredQuery results.

This Iterator loads the results lazily by evaluating the com.google.cloud.datastore.QueryResults and performing cursor queries.

The first query to the datastore is performed on creating an instance of the Iterator.

A call to hasNext() may cause a query to the Datastore if the current results page is fully processed.

A call to next() may not cause a Datastore query.

The remove() method throws an UnsupportedOperationException.
