public DatastoreStorageFactory.Builder setDatastore(Datastore datastore)

Assigns the Datastore to use for the DB interactions.

If the provided Datastore is configured with a namespace:

  • resulting single tenant storages will use the provided namespace;
  • resulting multitenant storages will concatenate the provided namespace with the tenant identifier. See setNamespaceConverter for more configuration.
