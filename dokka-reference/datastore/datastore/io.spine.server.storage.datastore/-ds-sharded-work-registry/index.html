DsShardedWorkRegistry

public class DsShardedWorkRegistry

A io.spine.server.delivery.ShardedWorkRegistry based on the Google Datastore storage.

The sharded work is stored as ShardSessionRecords in the Datastore. Each time the session is picked up, the corresponding Entity in the Datastore is updated.

It is recommended to use this implementation with Cloud Firestore in Datastore mode, as it provides the strong consistency for queries.
