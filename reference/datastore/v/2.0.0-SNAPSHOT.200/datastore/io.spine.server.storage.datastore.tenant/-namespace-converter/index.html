NamespaceConverter

@SPI
@Immutable
abstract class NamespaceConverter : Converter<String, TenantId>

A Converter for TenantId and the string Datastore namespace.

Define this converter in case if there is a need to override the default conversion behavior.

The implementation of the conversion must be commutative, meaning that the restored TenantId values should be equal to the initial.
