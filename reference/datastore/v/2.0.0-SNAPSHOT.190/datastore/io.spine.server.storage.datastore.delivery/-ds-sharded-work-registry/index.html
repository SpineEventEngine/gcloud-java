DsShardedWorkRegistry

open class DsShardedWorkRegistry : AbstractWorkRegistry, WithLogging

A io.spine.server.delivery.ShardedWorkRegistry based on the Google Datastore storage.

The sharded work is stored as ShardSessionRecords in the Datastore. Each time the session is picked up, the corresponding Entity in the Datastore is updated.

This storage uses transactions for read and write operations. It is also recommended using this implementation with Cloud Firestore in Datastore mode, as it enforces serializable isolation for transactions.
