DatastoreStorageFactory

open class DatastoreStorageFactory : StorageFactory, WithLogging

Creates Storages based on Datastore.

As a convenience API, provides an ability to configure the BoundedContextBuilders with the TenantIndex specific to the instance of Datastore configured for this factory.

As per design intention of StorageFactory, by default all storages created by this factory delegate the execution to instances of a pre-configured DsRecordStorage.
